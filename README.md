# estudiopropato.com.ar

## Setup

1. Instalar Node.js: https://nodejs.org/es/.
1. Instalar git: https://git-scm.com/download/win
1. En la linea de comando, correr: `git clone git@github.com:miparnisari/estudiopropato.git` para descargar este proyecto

## Subir cambios a la web

En la carpeta donde se descargo el proyecto:

1. En la linea de comando, correr: `npm install`.
1. En la linea de comando, correr: `gulp`. La web va a empezar a correr en http://localhost:3000/
